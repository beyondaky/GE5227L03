Map of Legal Gambling Sites and Help Centres in Singapore


Link >>> Legal Gambling Sites and Help Centres in Singapore


DESCRIPTION

Feeling lucky?

This ArcGIS API webmap provides a comprehensive directory of all legal gambling sites in Singapore, namely, Singapore Pools outlets, Horse Betting, Casinos and the neighbourhood Gaming Machine rooms. Users may geolocate their positions to find the nearest gambling site with the Locate Widget (third icon at top-left). Upon clicking a site of interest, a pop-up will appear containing name of outlet, address, type. Users may copy the address to the Direction Widget (bottom-left) to get the route. This webmap also features an option to switch basemaps and to toggle layer visibility.

If gambling problem gets out of hand, users should seek assistance by clicking on the 'Help' symbols. Information is provided.


DATASET

The datasets were created with Microsoft Excel and geocoded on ArcGIS Pro. The files were then shared on ArcGIS Online. The respective URLs were linked to Webmap.html. Picture-marker symbols were customised with web png images. Pop-ups containing Name of Outlet, Address and Type were also created with popuptemplate.
